Projects

I created a responsive personal portfolio website to showcase my skills, projects, and goals as I continue my studies in computer science. The website is designed to be fully responsive, adapting seamlessly to different screen sizes, from desktops to mobile devices.

Technologies Used: HTML, CSS

Features: Clean and modern layout. Fully responsive design, ensuring optimal viewing on any device. Smooth navigation and organized content sections: Home, About Me, Projects, Skills, and Contact. Simple yet effective use of CSS for layout and styling.

This project gave me the opportunity to apply my HTML and CSS skills in creating a functional and attractive web design. It also allowed me to learn more about creating responsive websites that provide a great user experience across devices.
